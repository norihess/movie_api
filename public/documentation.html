request url response
GET request /movies return array of all movies
GET request /movies/:Title return movie by it's title
GET request /genre/:Name return info about a specific movie genre
GET request /director/:Name return info about movie directors
POST request /users register new user to our system
PUT request /users/:Username Update the persons username info
POST request /users/:Username/movies/:MovieID Add a movie to a specific user's profile
DELETE request /users/:Username Unregister a user from the database
DELETE request /users/:Username/movies/:MovieID Delete a favorite movie from the users favorites
