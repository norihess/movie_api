request url response
GET request /movies return array of all movies
GET request /movies/:movieTitle return movie by it's title
GET request /genres/:genre return info about a specific movie genre
GET request /directors/:directorName return info about movie directors
POST request /register register to our system
PUT request /users/:ID/:infoToUpdate/:newValue Update the persons username info
POST request /users/:ID/favorites/:newFavorite Add a movie to a specific user's profile
DELETE request /users/:id/unregister Unregister a user from the database
DELETE request /users/:ID/favorites/:deleteFavorite Delete a favorite movie from the users favorites
